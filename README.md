## 介绍

云搭是一款功能强大、开源、免费的可视化搭建解决方案，它支持零代码搭建小程序、H5、问卷、图文等多种应用，我致力于提供一套简单、便捷、专业、可靠的多场景零代码平台。

技术栈前端为vue3，后端基于nodejs进行开发

## 开始

### 安装依赖

```bash
pnpm bootstrap
```

必须安装`pnpm`运行项目。

### 运行项目云搭平台

```bash
pnpm dev:dooring
```

### 运行项目云搭渲染器H5

```bash
pnpm dev:render
```
